# Bobu Ideas — Web UI
React + TypeScript + Vite application for the Bobu Ideas forum.

- Snapshot space: https://snapshot.box/?ref=bobu.ghost.io#/s:bobuthefarmer.eth
- Project site: ideas.bobu.exchange

## Quick Start
1) `npm i`
2) `npm run dev` → open the printed localhost URL

Build & preview:
- `npm run build`
- `npm run preview`

Environment:
- `VITE_APP_ENV=testnet` (default) or `VITE_APP_ENV=mainnet`

Notes:
- On Sepolia (testnet), a “Mint Testnet Bobu” helper is shown for connected wallets.
- New proposal creation redirects to the on-chain proposal page after confirmation.
